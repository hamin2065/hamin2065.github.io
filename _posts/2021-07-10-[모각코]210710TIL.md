# Start React Project

`create-react-app` 명령어를 통해서 프로젝트를 열어주었다. 

필요없는 파일들을 전부 삭제하고 src파일 안에 `App.js`와 `index.js`파일만 남겼다.

__CODE__

`App.js`
```js
import React from "react";

function App() {
  return <div>Hello!</div>
}

export default App;
```
`index.js`
```js
import React from 'react';
import ReactDOM from 'react-dom';

import App from './App';

ReactDOM.render(
  <React.StrictMode>
    <App />
  </React.StrictMode>,
  document.getElementById('root')
);
```
